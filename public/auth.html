📚 How JWT Authentication Works
  1. Register/Login: User provides credentials
  2. Server Verification: Server checks username/password
  3. Token Creation: Server creates JWT token with user info
  4. Token Storage: Client stores token (localStorage)
  5. Protected Requests: Client sends token in Authorization header
  6. Token Verification: Server verifies token on each request

Token Format: Bearer <jwt-token>
